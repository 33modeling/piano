# 꼬마 피아노 연습실

유아와 초보자를 위한 동요 중심 피아노 계이름 연습 웹앱입니다. 별도 빌드 없이 `index.html`을 브라우저에서 열면 실행됩니다.

## 기능

- 동요 선택: 작은 별, 나비야, 학교 종, 비행기, 리듬 노 젓기, 런던 다리, 브라더 존, 기쁨의 노래, 징글벨, 올드 맥도널드
- 곡 항목: 달팽이의 하루(작사 조원경 / 작곡 김진성)는 악보 입력용 항목으로 제공
- 터치 피아노: 휴대폰과 태블릿에서 가로 스크롤로 연주
- 계이름 연습: 도레미 표시, 실제 건반 터치, Web Audio 재생
- 실제 피아노 판정: 마이크 단음 감지와 Web MIDI 입력 지원
- 연습 스테이지: 듣기, 배우기, 연주 모드와 움직이는 악보 트랙
- 추천 시작: 현재 진행 상황에 맞춰 이어서 연습하거나 오답 복습으로 이동
- 어려운 음 복습: 틀린 목표 음을 모아 별도 연습곡으로 반복
- 큰 화면 모드: 연습 카드와 피아노에 집중하는 보기
- 색상 테마: 맑은 하늘, 비비드 팝, 무지개, 밤하늘
- 사운드 설정: 부드러운 피아노, 반짝이는 건반, 종소리, 신스 리드와 볼륨 조절
- 단계별 학습: 오른손, 왼손, 양손, 리듬, 중급 양손
- 손가락 안내: 오른손/왼손 번호와 손가락 이름 표시
- 계이름 생성: 한글 계이름, Do Re Mi, C D E, 숫자보 입력 지원
- 5분 루틴: 첫 5음, 동요 절반, 양손 단계 미션
- 진행 저장: 브라우저 `localStorage`에 성공 수, 단계, 속도 저장

## 악보 입력 예시

```text
도 도 솔 솔 라 라 솔:2 | 파 파 미 미 레 레 도:2
```

```text
C4 C4 G4 G4 A4 A4 G4:2 | F4 F4 E4 E4 D4 D4 C4:2
```

마디는 `|` 로 나눌 수 있고, `:2` 또는 `/2`처럼 박자 길이를 붙일 수 있습니다.

## 실행

브라우저에서 `index.html`을 열면 기본 연습 기능을 사용할 수 있습니다. 마이크/MIDI 판정까지 확인하려면 로컬 서버로 실행하는 편이 안정적입니다.

```bash
python3 -m http.server 4173
```

그 다음 `http://localhost:4173`에 접속합니다.

## Android APK 빌드

Android 프로젝트는 `android/` 폴더에 있습니다. 로컬 서명 설정은 `android/keystore.properties`와 keystore 파일을 사용하며, 두 파일은 git에 포함하지 않습니다.

```bash
cd android
JAVA_HOME=/home/kms/.local/jdk21 PATH=/home/kms/.local/jdk21/bin:$PATH ./gradlew assembleRelease
```

생성 APK:

```text
android/app/build/outputs/apk/release/app-release.apk
```
